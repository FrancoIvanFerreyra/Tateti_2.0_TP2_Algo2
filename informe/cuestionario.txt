¿Qué es un svn?
SVN es una herramienta de control de versiones centralizada utilizado para gestionar proyectos de desarrollo de software. Permite hacer un seguimiento de los cambios en los archivos y colaborar en equipo.

¿Que es una “Ruta absoluta” o una “Ruta relativa”?
Ruta absoluta: Es una dirección completa que especifica la ubicación de un archivo o directorio desde el directorio raíz del sistema de archivos.
Ruta relativa: Es una dirección de la ubicación de un archivo o directorio en relación con el directorio actual. No comienza desde la raíz del sistema de archivos.

¿Qué es git?
Git es un sistema de control de versiones distribuido que permite llevar un seguimiento de los cambios en el código fuente durante el desarrollo de un proyecto. A diferencia de los sistemas centralizados como SVN, Git permite que cada desarrollador tenga una copia local completa del repositorio, lo que facilita el trabajo offline y permite la creación de ramas de manera rápida y eficiente.
